PIPELINE_01 :: RAW INPUTS

Incident Feature Vectors

Every incident logged in Sahaay is converted into a structured feature vector before ingestion by the Random Forest models. The features consist of:

EVENT TYPE Planned vs. Unplanned
EVENT CAUSE 13 Target Categories
GEOGRAPHY Bangalore Centroid Nodes
TEMPORAL Hour / Day / Month

Temporal factors are converted into cyclical representations to capture peak vs. off-peak commute patterns. Corridor criticality mappings translate locations into topological impact parameters.

PIPELINE_02 :: ENSEMBLE REGRESSOR

Random Forest Regressor (Duration Model)

Predicts event resolution duration. The target variable is log-transformed to stabilize variance and handle extreme outlier delays:

\[\log(1 + \text{Duration}) = f_{\text{RF}}(X_{\text{features}})\] \[\text{Duration}_{\text{pred}} = e^{f_{\text{RF}}(X)} - 1\]
R² SCORE ~0.34
ESTIMATORS 150 Trees
MIN RESOLUTION 5.0 min (floor)
PIPELINE_03 :: ENSEMBLE CLASSIFIER

Random Forest Classifier (Road Closure Model)

Computes the binary probability that an incident will require a physical road closure, which triggers active barricading recommendations:

\[P(\text{Closure} = 1 \mid X) = \frac{1}{M} \sum_{m=1}^{M} T_m(X)\]
ROC-AUC SCORE 0.78
CRITERION Gini Impurity
FEATURES Type, Cause, Corridor

PIPELINE_05 :: FEEDBACK SYNAPSE

Closed-Loop Retraining & Bounds

Field outcomes logged inside the Learning tab trigger model retraining. Scikit-learn pipelines are refitted with new observations and reloaded in under 2 seconds.
